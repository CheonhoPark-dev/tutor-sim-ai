# TutorSim.AI - 제품 요구사항 문서 (PRD)

## 1. 제품 개요

### 1.1 제품 설명
TutorSim.AI는 사용자가 AI 챗봇에게 강의를 하며 학습 내용을 정리하고 피드백을 받는 자기주도형 교육 앱입니다. "가르치는 것이 최고의 학습이다"라는 원칙을 AI 기술과 결합하여 실전형 학습 환경을 제공합니다.

### 1.2 목표 사용자
- 수험생 (개념 정리 목적)
- 발표 연습이 필요한 학생
- 예비 교사 및 교육 전공자
- 외국어 학습자
- 직장인 프레젠터

### 1.3 핵심 가치 제안
- 실전형 교육 환경 제공
- AI 기반 즉각적 피드백
- 맞춤형 학습 경험
- 크로스 플랫폼 지원 (웹/앱)

## 2. 기술 요구사항

### 2.1 프론트엔드
- 웹: Next.js (React 기반)
  - PDF 뷰어 구현
  - 실시간 음성/텍스트 입력 처리
  - 반응형 디자인
  
- 앱: Flutter
  - Android/iOS 크로스 플랫폼 지원
  - 음성 녹음 및 실시간 처리
  - 오프라인 데이터 캐싱

### 2.2 백엔드
- Firebase 기반 구현
  - 사용자 인증
  - 실시간 데이터베이스
  - 파일 스토리지
  - 서버리스 함수

### 2.3 AI 통합
- Google Gemini 2.5 Flash API 연동
- Google Cloud Speech-to-Text API 음성 인식
- Text-to-Speech API 음성 출력

## 3. 핵심 기능 요구사항

### 3.1 강의 기능
- 실시간 음성/텍스트 입력 지원
- 강의 내용 자동 저장
- 실시간 텍스트 변환 (STT)
- 강의 일시정지/재개 기능

### 3.2 가상 학생 시스템
초기 구현 성향 (Phase 1):
- 호기심 많은 학생
- 무관심한 학생

교육 수준:
- 중학교 (고정)

향후 확장 예정 성향:
- 부정적인 학생
- 긍정적인 학생
- 적극적인 학생

향후 확장 예정 교육 수준:
- 초등학교
- 고등학교
- 대학교
- 박사과정

### 3.3 학습 자료 관리
- PDF, 이미지 파일 업로드
- 기본 문서 표시 기능
- 실시간 참고 자료 뷰어
- (향후 구현) 문서 자동 분석
- (향후 구현) 자료 기반 질문 생성

### 3.4 피드백 시스템
실시간 피드백:
- 발화 속도 분석
- 전문용어 사용 빈도
- 설명 충실도 평가

종합 피드백:
- 강의 요약본 생성
- 개선점 제안
- 학생 반응 분석

### 3.5 실시간 가이드
- 침묵 감지 (5초 기준)
- 문맥 기반 프롬프트 제공
- 추가 설명 예시 제공

## 4. 비즈니스 요구사항

### 4.1 기본 기능 (무료)
- 기본 강의 기능
- 제한된 가상 학생 성향
- 기본 피드백

### 4.2 프리미엄 기능 (유료)
- 상세 분석 리포트
- 추가 가상 학생 성향
- 고급 피드백 기능
- 강의 녹화/저장

### 4.3 구독 모델
- 월간 구독제
- 연간 구독제 (할인)
- 교육기관 단체 라이선스

## 5. 보안 요구사항

### 5.1 데이터 보안
- 사용자 데이터 암호화
- 강의 내용 보안 저장
- 개인정보 보호 준수

### 5.2 접근 제어
- 역할 기반 접근 제어
- API 키 보안 관리
- 세션 관리

## 6. 성능 요구사항

### 6.1 응답 시간
- 음성 인식: 1초 이내
- AI 응답: 1.5초 이내 (Gemini Flash API 기준)
- 파일 업로드: 5초 이내 (10MB 기준)

### 6.2 동시 접속
- 최소 1000명 동시 접속 지원
- 실시간 처리 지연 최소화

### 6.3 확장성
- 마이크로서비스 아키텍처 지원
- 자동 스케일링 구현

## 7. 출시 계획

### 7.1 Phase 1 (MVP)
- 기본 강의 기능
- 텍스트 기반 상호작용
- 기본 피드백 시스템
- 2가지 가상 학생 성향 (호기심/무관심)
- 중학교 수준 고정
- 기본 문서 표시 기능
